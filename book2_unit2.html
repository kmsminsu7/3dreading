만약 네가 보물을 찾는다면, 우리와 나눠줘.
🟡 If you find the treasure

💡 if 부사절이 앞 → 쉼표 필수! / 현재 시제(find)로 미래 조건 표현!
✨ 접속사만 바꾸면 뜻이 달라져요!
같은 문장, 접속사만 바꾸면 완전히 다른 의미!
because Ha-on was happy because she found the key. (열쇠를 찾았기 때문에 행복했다)
when Ha-on was happy when she found the key. (열쇠를 찾았을 때 행복했다)
if Ha-on will be happy if she finds the key. (만약 열쇠를 찾는다면 행복할 것이다)
💡 핵심: 접속사 하나가 바뀌면 이유·시간·조건이 달라져요. 독해할 때 접속사를 먼저 확인하는 습관이 중요해요! 🎯
✏️ 빈칸 채우기 — 어떤 접속사가 맞을까요?
1. ______ it was dark outside, they could not read the map. (어두웠기 때문에)
2. ______ you work hard, you will find the treasure. (열심히 하면)
3. Luna screamed ______ she saw a strange light. (이상한 빛을 봤을 때)
✅ 정답: 1. Because   2. If   3. when

지문 독해 📖
선장의 항해 일지 — 부사절을 찾아라!
5
📌 노란 꺾쇠 { } = 부사절 / 접속사 종류: when · because · if
⚓ 선장의 항해 일지 — Captain's Log
📄 영어 지문
① {When the storm hits}, all passengers must stay inside the cabin.
🟡 when절 — 앞마당, 쉼표(,) 필수
② We cannot see the lighthouse {because the fog is too thick}.
🟡 because절 — 뒤마당, 쉼표 없음
③ {If you see a red signal light}, come to the deck immediately.
🟡 if절 — 앞마당, 쉼표 필수
④ The ship slowed down {when the captain spotted the rocks}.
🟡 when절 — 뒤마당, 쉼표 없음
⑤ Luna was not afraid {because she had found a life jacket}.
🟡 because절 — 뒤마당, 쉼표 없음
⑥ {If the weather clears up}, we will reach Waldo Island by morning.
🟡 if절 — 앞마당, 쉼표 필수
📖 한글 해석
①폭풍이 몰아칠 때, 모든 승객은 선실 안에 있어야 한다.
②안개가 너무 짙기 때문에 우리는 등대를 볼 수 없다.
③만약 빨간 신호등이 보이면, 즉시 갑판으로 나와라.
④선장이 암초를 발견했을 때 배가 속도를 줄였다.
⑤루나는 구명조끼를 발견했기 때문에 무섭지 않았다.
⑥만약 날씨가 개면, 우리는 아침까지 월도 섬에 도착할 것이다.

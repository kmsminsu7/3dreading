📌 핵심 규칙 — 명사블럭이란?
명사블럭 = 대장 명사 + 꾸며주는 친구. 주어·목적어 자리에 들어가요.
📌 3가지만 기억해요!
① 명사 혼자만 있는 게 아니에요
the book → the book [on the desk] (책상 위의 책)
② 친구는 딱 4가지 — 전·투·분·관
전치사구 / to부정사 / 분사구 / 관계사절 → Unit 2~5에서 하나씩 자세히!
③ 친구를 걷어내면 뼈대(대장 명사)가 보여요!
[The cat with white paws] chased [the ball on the table]
→ 뼈대: cat · chased · ball
📐 짧은 문장 vs 긴 문장 — 뼈대는 똑같아요!
❶ 짧은 문장
The cat chased the ball.
간단해요! 🙂
❷ 긴 문장 — 친구 등장!
The cat [with white paws] chased the ball [on the table].
with white paws
on the table
뼈대는 똑같아요! 💪
📝 단어장 chased 쫓았다 · paws 동물 발 · skeleton 뼈대

📌 핵심 규칙 — 전치사구란?
전치사 + 명사 = 전치사구. 명사 뒤에서 위치·방향·시간·관계를 알려줘요!
전치사구 = 전치사 + 명사
in + the gym = in the gym  (체육관 안에)
🗂️ 자주 쓰는 전치사 8가지
전치사 뜻 예시 듣기
in~ 안에the box in the room
on~ 위에the book on the desk
under~ 아래에the cat under the chair

📌 핵심 규칙 — to부정사구란?
to + 동사원형이 명사 뒤에 붙어서 "~할" 또는 "~하는"의 뜻으로 명사를 꾸며줘요!
to부정사구 = to + 동사원형
a code  to unlock  = a code [to unlock]   (열어야 할 코드)
📌 3가지만 기억해요!
① to + 동사원형 = to부정사구
a book [to read] → read가 동사원형! ✅
② 앞에 있는 명사를 "~할/하는"으로 꾸며줘요
something [to eat] → 먹을 것 (something = 무언가)
a way [to solve it] → 해결할 방법
③ 전치사구(전)와 헷갈리지 않기!
🟠 전치사구
a book [on the desk]
on = 전치사 (명사만 옴)
🔴 to부정사구
a book [to read]
to = ~할 (동사가 옴)
하온
아하! on the desk는 전치사구(on = 전치사), to read는 to부정사구(to + 동사)! 뒤에 뭐가 오느냐로 구분하면 되는 거지? 😄

"~하는" · "~하고 있는"
(능동 — 스스로 하는 것)
the boy standing
→ 서 있는 소년
a shadow moving
→ 움직이는 그림자
과거분사 (-ed/pp)
"~된" · "~당한"
(수동 — 당하는 것)
something hidden
→ 숨겨진 것
a painting damaged
→ 손상된 그림

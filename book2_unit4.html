2
📌 명사절이란? — 절이 명사처럼 쓰여요!
명사 → 문장에서 주어·목적어·보어 역할
명사절 → 절 전체가 주어·목적어·보어 역할! 🟣
예: I know the answer. (명사)
    I know [that the key is here]. (명사절 — 절 전체가 목적어!)
명사절 접속사 3총사
that
~라는 것
사실 전달
if
~인지
Yes/No 의문
whether
~인지 아닌지
선택 의문
💡 부사절 if vs 명사절 if: 조건(만약~라면)이면 부사절! 의문(~인지)이면 명사절!
If you work hard, you'll succeed. (부사절 — 조건)
I don't know if he will come. (명사절 — ~인지)
📌 명사절의 3가지 역할
① 주어 자리
That she is smart
is clear.
그녀가 똑똑하다는 것이 분명하다.
② 목적어 자리
I know
that she is smart.
나는 그녀가 똑똑하다는 것을 안다.
③ 보어 자리
The truth is
that she is smart.
진실은 그녀가 똑똑하다는 것이다.
⭐ 가주어 It ~ that 패턴
명사절이 주어 자리에 오면 길어서 어색해요.
→ It을 앞에 두고 that절을 뒤로 보내요!
원래: [That the key is here] is important.
→ It is important [that the key is here].
It = 가주어(가짜 주어) / that절 = 진주어(진짜 주어)

예문 분석 A 🔍
that절 — 목적어·보어·가주어 다이어그램
3
that — 목적어 자리 (가장 많이 쓰여요!)
🔊
Seo-jun knows [that the map shows a real island].

🟣 that the map shows a real island

that — 보어 자리
🔊
The truth is [that my grandfather hid the treasure here].
진실은 나의 할아버지가 여기에 보물을 숨겼다는 것이다.
→
→
🟣 that my grandfather hid the treasure
🟣 that절이 보어(분홍 헥사곤) 자리에!
⭐ 가주어 It ~ that 패턴
🔊
It is important [that we solve this code before sunrise].
우리가 해가 뜨기 전에 이 암호를 푸는 것이 중요하다.
→
→
진주어 → 🟣 that we solve this code before sunrise

Unit 4 · p.4 · [명사절]
예문 분석 B 🔍
if / whether — 의문을 나타내는 명사절
4
if — "~인지" (명사절, 목적어 자리)
🔊
Luna asked [if the ancient code was a number].
루나는 고대 암호가 숫자인지 물었다.
→
→
🟣 if the code was a number
💡 if = "~인지" — Yes/No로 대답할 수 있는 의문! 목적어 자리!
whether ~ or not — "~인지 아닌지"
🔊
Ha-on wondered [whether the code was a number or a word].
하온은 암호가 숫자인지 단어인지 궁금해했다.
→
→
🟣 whether the code was a number or a word
💡 whether = "A인지 B인지" — 두 가지 선택! or not/or ~ 함께 자주 사용!
✨ that vs if vs whether — 핵심 비교
that I believe that the key is here.
사실을 전달 — "열쇠가 여기 있다는 것을"
if I wonder if the key is here.
Yes/No 의문 — "열쇠가 여기 있는지"
whether I wonder whether the key is here or not.
선택 의문 — "있는지 없는지"
💡 꿀팁: 동사 다음에 나오면 → that(사실) / if(~인지) / whether(~인지 아닌지) 중 의미에 맞는 것을 골라요!

📌 보라 꺾쇠 { } = 명사절 / 접속사: that · if · whether
📜 두루마리 암호문 — The Ancient Scroll
📄 영어 지문
① Seo-jun realized {that the symbols on the wall were a secret code}.
🟣 that절 — 목적어 자리, "기호들이 암호였다는 것을"
② Ha-on wondered {whether the treasure was gold or jewels}.
🟣 whether절 — 목적어 자리, "금인지 보석인지"
③ It is clear {that the explorer left this message for his family}.
🟣 It~that 가주어 패턴 — "탐험가가 메시지를 남겼다는 것이 분명하다"
④ Luna asked {if the door at the end of the room was the exit}.
🟣 if절 — 목적어 자리, "방 끝 문이 출구인지"
⑤ The scroll said {that the final key was inside the treasure box itself}.
🟣 that절 — 목적어 자리, "최종 열쇠가 보물 상자 안에 있다는 것을"
⑥ It is true {that your grandfather was the greatest explorer on the island}.
🟣 It~that 가주어 패턴 — "할아버지가 섬 최고의 탐험가였다는 것이 사실이다"
📖 한글 해석
①서준은 벽의 기호들이 비밀 암호였다는 것을 깨달았다.
②하온은 보물이 금인지 보석인지 궁금해했다.
③탐험가가 가족을 위해 이 메시지를 남겼다는 것이 분명하다.
④루나는 방 끝의 문이 출구인지 물었다.
⑤두루마리에는 최종 열쇠가 보물 상자 안에 있다고 쓰여 있었다.
⑥네 할아버지가 그 섬 최고의 탐험가였다는 것은 사실이다.

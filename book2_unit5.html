핵심 규칙 📌
관계대명사 주격·목적격·소유격 한눈에!
2
📌 관계대명사 — 명사를 뒤에서(!) 꾸며주는 절
💡 공식: 선행사(꾸밈 받는 명사) + [관계대명사 + S+V...]
the boy [who runs fast] → "빨리 달리는 소년" → boy를 뒤에서 꾸며줌!
① 주격 — 관계사절 안에서 주어 역할
who
사람 선행사
the explorer who was born here
which
사물 선행사
the map which shows the route
that
사람·사물 모두 OK (구어체)
the treasure that our family hid / the explorer that found it
② 목적격 — 관계사절 안에서 목적어 역할
the explorer [whom] Ha-on found
= 하온이 발견한 탐험가 (found의 목적어가 whom)
→ whom/which/that 사용 가능 / 구어체에선 생략도 OK!
③ 소유격 whose — "~의"
whose
사람·사물 모두 사용
the explorer whose map we found
("우리가 찾은 지도의 탐험가" → 탐험가의 지도)
📌 선택 공식: 선행사가 사람 → who/whom/whose / 사물 → which/whose / 둘 다 → that

목적격 — 관계사절 안에서 목적어 자리
🔊
The map [that Ha-on found] had three red marks.
하온이 발견한 지도에는 세 개의 빨간 표시가 있었다.

주격the map [that shows the route] → that 뒤 바로 동사! (that = 주어)
목적격the map [that Ha-on found] → that 뒤 주어+동사! (that = 목적어)
🟢 that Ha-on found

✏️ 문장 합치기 훈련
두 문장을 관계대명사로 연결해봐요!
① I met a girl. + She found the key.
→ I met a girl [who found the key].
② The treasure was hidden. + Ha-on's grandfather hid it.
→ The treasure [that Ha-on's grandfather hid] was hidden.
③ We found an explorer. + His map was very old.
→ We found an explorer [whose map was very old].
💡 합치기 공식: 두 번째 문장의 명사(사람/사물)를 관계대명사로 바꾸고 선행사 바로 뒤에 붙여요!

📌 초록 꺾쇠 { } = 관계대명사절 / 종류: who · which/that · whose
🗺️ 보물지도의 단서들 — Clues on the Treasure Map
📄 영어 지문
① Find the lighthouse keeper {who has lived on the island for fifty years}.
🟢 who절 — 주격, lighthouse keeper를 꾸밈 / 어떤 등대지기? → 50년간 살아온
② The stone door {that guards the final room} will open only at midnight.
🟢 that절 — 주격, stone door를 꾸밈 / 어떤 돌문? → 마지막 방을 지키는
③ Use the golden key {which Ha-on found in the secret room}.
🟢 which절 — 목적격, key를 꾸밈 / 어떤 열쇠? → 하온이 비밀 방에서 찾은
④ Look for the passage {whose entrance is behind the waterfall}.
🟢 whose절 — 소유격, passage를 꾸밈 / 어떤 통로? → 입구가 폭포 뒤에 있는
⑤ The treasure {that our family has guarded for generations} is inside.
🟢 that절 — 목적격, treasure를 꾸밈 / 어떤 보물? → 우리 가문이 대대로 지켜온
⑥ The one {who truly deserves the treasure} will know what to do.
🟢 who절 — 주격, one을 꾸밈 / 어떤 사람? → 보물을 진정으로 받을 자격이 있는
📖 한글 해석
①50년간 그 섬에 살아온 등대지기를 찾아라.
②마지막 방을 지키는 돌문은 자정에만 열릴 것이다.
③하온이 비밀 방에서 찾은 황금 열쇠를 사용해라.
④입구가 폭포 뒤에 있는 통로를 찾아라.
⑤우리 가문이 대대로 지켜온 보물이 안에 있다.
⑥보물을 진정으로 받을 자격이 있는 사람이 무엇을 해야 할지 알 것이다.
